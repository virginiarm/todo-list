# React + Vite
# project name : 
  Todo list
# Description 
This is a todo, a productivity tool to maximize time and organize your priorities.
## Installation Instructions
Follow these steps to install and set up the project locally:
 1. clone the repository
 git clone https://github.com/virginiarm/todo-list.git
 2. navigate to the project directory
 3. npm install
 4. start the developement server by npm start
 and then run on browser.
